Sphinx Game

A Web3 carnival game built in Typescript and Solidity. Upon contract deployment, users could enter the website and try to solve riddles for an entry free denoted in Ethereum. The first person to submit three correct answers to the riddles gains 69% of the total pot. Web hooks using RainbowKit + wagmi packages.
